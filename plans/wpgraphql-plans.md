# WPGraphQL Plans

Source: https://www.wpgraphql.com/ | https://wordpress.org/plugins/wp-graphql/ | https://opencollective.com/wp-graphql

## Overview

WPGraphQL is a free, open-source WordPress plugin with no paid licensing tiers. The core plugin is available at no cost under the GNU General Public License (GPL v2 or later). Commercial support and extensions exist through third-party providers and the ecosystem.

## Core Plugin (Free)

- **Price:** $0
- **License:** GPL v2 or later (open source)
- **Source:** Available on WordPress.org and GitHub (github.com/wp-graphql/wp-graphql)
- **Features:**
  - Full GraphQL API for all WordPress core data types
  - Posts, pages, custom post types, taxonomies, users, menus, media
  - Queries and mutations
  - Cursor-based pagination (up to 100 nodes per request)
  - Built-in GraphiQL IDE in WordPress dashboard
  - Extendable schema via PHP hooks
  - Authentication support (nonces, application passwords)
  - WPGraphQL Smart Cache compatibility
  - Active community support via GitHub Discussions

## Open Collective (Sponsorship)

WPGraphQL accepts financial contributions through Open Collective to fund ongoing development.

- **URL:** https://opencollective.com/wp-graphql
- **Backer tiers:** Various (individual contributor to corporate sponsor)
- **Benefits:** Recognition on the WPGraphQL website and Open Collective page; priority consideration for feature requests
- **Corporate Sponsor:** Automattic is the primary sponsor and employs the creator, Jason Bahl, to work on WPGraphQL full-time

## Hosting Considerations

WPGraphQL runs on any self-hosted WordPress installation. Costs are entirely tied to WordPress hosting:

- **Shared hosting:** $3–$30/month (performance may limit GraphQL scalability)
- **Managed WordPress (WP Engine, Pressable, Kinsta):** $25–$400+/month with better GraphQL support
- **Headless hosting (Pantheon, WP Engine Headless):** Custom pricing with optimized WPGraphQL infrastructure

## Notes

- No per-request fees, no API key required, no usage caps imposed by WPGraphQL itself
- Rate limits depend entirely on WordPress hosting configuration
- The plugin is maintained by Automattic as a Canonical Plugin, ensuring long-term stability
